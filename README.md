# Contact Management System (Python)

A simple **command-line contact management system** written in Python.
This project is beginner-friendly and demonstrates the use of:
- Dictionaries
- Loops
- Conditional statements
- Exception handling

## Features
- Add new contacts
- Search contact numbers
- Update existing contacts
- Delete contacts
- View all saved contacts

## Requirements
- Python 3.x

## How to Run
```bash
python PhoneBook.py
